# Dissertation-Project
Big Data ETL Pipeline on Cloud Platform
Η παρούσα εργασία επικεντρώνεται στην ανάπτυξη ενός ολοκληρωμένου Data Pipeline σε περιβάλλον cloud για την αυτοματοποίηση της εξαγωγής, μετασχηματισμού και φόρτωσης δεδομένων (ETL) σε έναν κεντρικό προορισμό. Η υλοποίηση πραγματοποιήθηκε με τη χρήση Apache Spark (PySpark) και υπηρεσιών του Google Cloud Platform (GCP). Το σύστημα βασίζεται στις αρχιτεκτονικές Medallion Architecture και Champion vs Challenger, με στόχο την οργάνωση των δεδομένων και τη συνεχή αξιολόγηση μοντέλων μηχανικής μάθησης. Επιπλέον, πραγματοποιήθηκε οπτικοποίηση δεδομένων για τη διευκόλυνση της ανάλυσης και της ερμηνείας των αποτελεσμάτων.
## Τεχνολογίες
Apache Spark (PySpark)
Google Cloud Platform (GCP)
Apache Airflow
Google Cloud Storage
PySpark MLlib
## Δυνατότητες
Αυτοματοποιημένη διαδικασία ETL
Υλοποίηση Data Pipeline σε cloud περιβάλλον
Επεξεργασία και μετασχηματισμός δεδομένων
Εκπαίδευση και αξιολόγηση μοντέλων μηχανικής μάθησης
Ενορχήστρωση ροών εργασίας με Apache Airflow
Οπτικοποίηση και ανάλυση δεδομένων
